🛡️ DDoS Protection System
Автономная система защиты от DDoS-атак с биометрической аутентификацией через Telegram бота. Система мониторит сетевой трафик, автоматически блокирует атакующие IP-адреса и отправляет уведомления администраторам.

✨ Возможности
🔍 Мониторинг трафика в реальном времени на порту 80

🚨 Автоблокировка IP при превышении 1000 пакетов/секунду

📱 Уведомления в Telegram о блокировках

🔐 Биометрическая аутентификация по лицу

⏱️ Авторазблокировка через 60 минут

🐳 Полная контейнеризация через Docker

🚀 Быстрый старт
1. Предварительные требования
Docker и Docker Compose

Сервер с Linux (рекомендуется VPS)

Telegram бот (создать через @BotFather)

2. Настройка
Клонируйте и настройте проект:

bash
git clone <repository-url>
cd ddos-protection-system
Отредактируйте docker-compose.yml:

yaml
environment:
  - BOT_TOKEN=your_bot_token_here
  - ADMIN_CHAT_ID=your_chat_id_here
Где взять:

BOT_TOKEN - получите у @BotFather

ADMIN_CHAT_ID - узнайте через @userinfobot

3. Запуск
bash
docker-compose build --no-cache
docker-compose up -d
4. Регистрация администратора
Найдите бота в Telegram по имени

Отправьте команду: /start

Добавьте первого администратора: /add_first_admin

Отправьте фото вашего лица

Готово! Вы зарегистрированы как администратор

5. Аутентификация
Для доступа к командам используйте:

text
/login - аутентификация по лицу
📋 Команды бота
Основные команды:
/stats - статистика блокировок

/blocked - список заблокированных IP

/unblock <IP> - разблокировать IP

/status - статус системы

Административные команды:
/add_admin - добавить администратора

/list_admins - список администраторов

/remove_admin <ID> - удалить администратора
